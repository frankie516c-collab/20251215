#라벨에 대하여
```HTML
<!DOCTYPE html>
<html lang="ko">
<head>
	<meta charset="UTF-8">
	<meta name="viewport" content="width=device-width, initial-scale=1.0">
	<link rel="icon" href="../../images/logo.png"/>
	<title>form</title>
</head>
<body>
	<h1>form</h1>
  <form action="result.html" method="get">
    <label for="userId">아이디 : </label>
    <input type="text" name="userId" id="userId">
    <input type="submit" value="인증요청">
  </form>
</body>
</html>
```
- input과 label이 하나의 쌍이다
    - **중요: label for="userId"랑 id="userId"를 동일하게 해야됨**
    - 같이 묶여있기 때문에 칸(input)이랑 아이디(label) 눌렀을 때 둘 다 칸안에 글자를 쓸 수 있음
    - 하나의 쌍이니까 div로 묶기도 함 (필수는 아님. 가독성 목적)
<div>
<label for="userId">아이디 : </label>
<input type="text" name="userId" id="userId">
</div>

```HTML
<!DOCTYPE html>
<html lang="ko">
    <head>
        <meta charset="UTF-8">
        <meta anme="viewport" content="width=device-width, initial-scale=1.0"
        <title>폼태그</title>
    </head>
    <body>
        <form>
          <label for="txt">글: </label>  
          <input type="text" name="txt" id="txt" required placeholder="글 작성해 주세요">
            <button type="submit">제출</button>
        </form>
    </body>
</html>
```
- label, id, name 다 txt지만 묶인건 label과 id 뿐이고 name은 프로토콜에 나오는 글자이기 때문에 다른 역할을 함
